#### **VS Code Tasks Integration**

**Create `.vscode/tasks.json`:**
```json
{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "Run Diagnostic",
      "type": "shell",
      "command": "node universal-diagnostic.js",
      "problemMatcher": []
    }
  ]
}
```

**Then:**
- Press `Ctrl + Shift + P` (Mac: `Cmd + Shift + P`)
- Type "Run Task"
- Select "Run Diagnostic"
- Done!

#### **Package.json Script**

**Add to `package.json`:**
```json
{

  }
}
```

**Then run:**
```bash

```
