## Overview

Advanced Knock Mobile is a React Native/Expo mobile application for door-to-door sales teams. Features include offline-first architecture, knock logging, lead management, territory intelligence, and AI coaching.

---

## Tech Stack

- **React Native:** 0.81.5
- **Expo:** ~54.0.0
- **TypeScript:** 5.3.0
- **Zustand:** ^4.5.0 (State Management)
- **React Navigation:** ^6.1.9
- **React Native Maps:** 1.20.1
- **AsyncStorage:** 2.2.0 (Offline-first)

## Quick Start

### Prerequisites

- Node.js 20+
- npm or pnpm
- Expo CLI
- iOS Simulator or Android Emulator (for development)

### Installation

```bash
# Clone repository
git clone https://github.com/Advanced-Knock/advanced-knock-mobile.git
cd advanced-knock-mobile

# Install dependencies
npm install

# Start development server
npx expo start
```

---

## Development

### Running the App

```bash
# Start Expo development server
npx expo start

# Run on iOS
npx expo start --ios

# Run on Android
npx expo start --android
```

### Building

```bash
# Build for development
npm run build

# Build for production
npm run build:production
```
